# text-case-converter-plus

## Описание
Browser extension для конвертации регистра текста в сайдпанели браузера.

## Репозиторий
- GitHub: git@github.com:bazzzdev/text-case-converter-plus.git
- Рабочая ветка: main

## Стек
- Browser Extension (Manifest V3)

## Быстрый старт
1. Открыть chrome://extensions/.
2. Включить Developer mode.
3. Нажать Load unpacked и выбрать папку chrome/.
4. Для Firefox использовать сборку из релизов (XPI).

## Структура
- chrome/
- firefox/
