A convenient browser extension for quick text case conversion.

## Features

- Convert text between different cases
- Quick access via context menu
- Support for multiple text cases
- Simple and user-friendly interface

## Installation

### Google Chrome / Microsoft Edge

1. Download the `text-case-converter-plus-chrome-1.0.0-release.zip` file from [Releases](../../releases)
2. Extract the zip archive to a folder
3. Open your browser and navigate to:
   - **Chrome**: `chrome://extensions/`
   - **Edge**: `edge://extensions/`
4. Enable Developer mode in the top right corner
5. Click "Load unpacked" button
6. Select the extracted extension folder
7. Extension installed! âœ“

### Mozilla Firefox

1. Download the `text-case-converter-plus-firefox-1.0.0-release.xpi` file from [Releases](../../releases)
2. Open the Firefox browser
3. Drag and drop the `.xpi` file into the browser window
   - Or: Open the browser menu (three dots in the top right corner)
   - Select "Add-ons and themes"
   - Click the gear icon in the top right and select "Install add-on from file..."
   - Select the downloaded `.xpi` file
4. Confirm the installation in the dialog that appears
5. Extension installed! âœ“

## Usage

1. Highlight text on any web page
2. Right-click on the selected text
3. Select "Text Case Converter Plus" from the context menu
4. Choose the desired conversion type
